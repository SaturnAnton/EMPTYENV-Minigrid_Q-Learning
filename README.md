# Q-Learning in MiniGrid Empty Environment

This repository contains a simple, from-scratch implementation of the **Tabular Q-Learning** algorithm applied to the `MiniGrid` environment (specifically the Empty rooms, e.g., `MiniGrid-Empty-8x8-v0`). 

The project is split into two main phases:
1. **Training:** The agent explores the environment, updates its Q-table, and learns the optimal policy to reach the green goal square.
2. **Visualization:** The trained agent is deployed in the environment, rendering its learned behavior step-by-step.
